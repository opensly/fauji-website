    Getting Started

    Introduction

    Fauji is a lightweight, zero-config testing framework for JavaScript and TypeScript. It provides Jest-like global APIs, powerful matchers, spies/stubs/mocks, and a simple CLI for running and watching tests. TypeScript users get full typings and autocompletion out of the box.

    Installation

    npm install fauji --save-dev
    No configuration required. Just write your tests and run them!

    First Test Example

                
              describe('Math', () => {
                beforeAll(() => { /* setup */ });
                beforeEach(() => { /* per-test setup */ });

                test('adds', () => {
                  expect(1 + 1).toBe(2);
                });

                test('object equality', () => {
                  expect({ a: 1, b: 2 }).toEqual({ a: 1, b: 2 });
                });

                afterEach(() => { /* per-test teardown */ });
                afterAll(() => { /* teardown */ });
              });

              run(); // Call at the end of your test file

    Running Tests

    Use the CLI to run your tests:

    npx fauji --dir ./tests --pattern test.js --name math --watch
    • --dir or -d: Test directory (default: current directory)
    • --pattern or -p: Test file pattern (default: test.js)
    • --name or -n: Filter test files by name
    • --watch or -w: Watch mode (reruns tests on file changes)
