Setup & Teardown

Test Hooks

Fauji provides hooks to run setup and teardown logic before or after tests and suites. These are useful for initializing state, cleaning up resources, or resetting mocks.

  • beforeAll(fn): Run once before all tests in a suite
  • afterAll(fn): Run once after all tests in a suite
  • beforeEach(fn): Run before each test in a suite
  • afterEach(fn): Run after each test in a suite

Example

describe('Array', () => {
  let arr;

  beforeAll(() => {
    // Runs once before all tests
    arr = [];
  });

  beforeEach(() => {
    // Runs before each test
    arr.push(1);
  });

  afterEach(() => {
    // Runs after each test
    arr.pop();
  });

  afterAll(() => {
    // Runs once after all tests
    arr = null;
  });

  test('length is 1 after push', () => {
    expect(arr).toHaveLength(1);
  });
});

Tips

  • Hooks are scoped to the current describe block.
  • You can nest describe blocks for more granular setup/teardown.
  • Use hooks to reset state, mock timers, or clean up resources.
