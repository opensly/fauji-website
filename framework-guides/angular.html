Testing Angular Apps with Fauji

Fauji is a lightweight, Jest-like testing framework for JavaScript/TypeScript. You can use it to test Angular components, services, and utilities with a familiar API and fast feedback loop.

1. Setup

  • Install Fauji as a dev dependency:
npm install --save-dev fauji
  • Ensure your test files are named with .test.js or .test.ts (or use your preferred pattern).
  • No need to import describe, test, expect, or hooksâ€”they are available as globals.

2. Example: Testing an Angular Service

// math.service.ts
export class MathService {
  add(a: number, b: number) { return a + b; }
}

// math.service.test.ts
describe('MathService', () => {
  let service;
  beforeAll(() => { service = new MathService(); });

  test('adds numbers', () => {
    expect(service.add(2, 3)).toBe(5);
  });
});

run();

3. Example: Testing an Angular Component

// counter.component.ts
export class CounterComponent {
  count = 0;
  increment() { this.count++; }
}

// counter.component.test.ts
describe('CounterComponent', () => {
  let comp;
  beforeEach(() => { comp = new CounterComponent(); });

  test('increments count', () => {
    comp.increment();
    expect(comp.count).toBe(1);
  });
});

run();

4. Using Matchers

  • Fauji provides toBe, toEqual, toBeTruthy, toThrow, and more. See the Matchers Guide.
  • Async support: await expect(Promise.resolve(5)).resolves.toBe(5)

5. Spies, Stubs, and Mocks

const { spy, stub, mock } = require('fauji');

describe('Spy example', () => {
  test('tracks calls', () => {
    const fn = spy((x) => x * 2);
    fn(2);
    expect(fn.callCount).toBe(1);
    expect(fn.calls[0][0]).toBe(2);
  });
});

6. Running Tests

npx fauji --dir ./src --pattern test.ts --watch
  • Use --dir to specify your test directory.
  • Use --pattern to match test files.
  • Use --watch for watch mode.

7. Angular-Specific Tips

  • For dependency injection, manually instantiate services or use Angular's TestBed if needed, then test with Fauji's matchers.
  • For DOM testing, use @angular/platform-browser utilities to query and interact with elements, then assert with Fauji.
  • Use Fauji's hooks (beforeAll, beforeEach, etc.) for setup/teardown.

8. TypeScript Support

  • Fauji provides typings for all matchers and hooks. Works out of the box in Angular projects.
